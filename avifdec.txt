avifdec.exe is not distributed with this repository.

Download the latest version of libavif for Windows from the official releases page:
https://github.com/AOMediaCodec/libavif/releases

1. Download the Windows archive (e.g., libavif-v1.3.0-windows-x64-dynamic.zip).
2. Extract avifdec.exe from the archive.
3. Copy avifdec.exe to this project folder.
