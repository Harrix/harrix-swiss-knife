По названию продукта или блюда оцени калорийность и определи, напиток это или еда.

Верни **одну строку** в формате TSV (столбцы через табуляцию, без заголовков, без markdown):

```text
Calories	Mode	Drink	Weight
```

- **Calories** — число (ккал):
  - при `Mode=weight` — калорийность **на 100 г** (типичное значение для продукта);
  - при `Mode=portion` — калории **за всю порцию**.
- **Mode** — режим ввода:
  - `weight` — если из названия неясно, что указана готовая порция (используй по умолчанию);
  - `portion` — если в названии явно порция, объём напитка, «за порцию», «180 мл», «стакан», «чашка» и т.п.
- **Drink** — `yes` для напитков (кофе, чай, сок, вода, молоко, лимonade и т.п.), иначе `no`.
- **Weight** — вес порции в граммах (целое). Если из названия можно извлечь граммы или миллилитры — укажи (мл ≈ г). Если вес неизвестен — `0`.

Примеры:

```text
165	weight	no	0
85	portion	yes	180
350	weight	no	0
52	weight	yes	0
```

(Для «Куриная грудка» → `165	weight	no	0`; для «Капучино 180 мл» → `85	portion	yes	180`.)

Название продукта:

```text
{{FOOD_NAME}}
```

Верни только одну строку TSV (4 столбца), без пояснений.
